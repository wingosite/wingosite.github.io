## 克隆并初始化项目

```bash
# 克隆项目
$ git clone https://gitee.com/bronson/v3-nest-admin

# 进入项目目录
$ cd v3-nest-admin

# 安装依赖
$ npm install
```

## 运行项目

```bash
# 开发环境
$ npm run start

# 监听模式
$ npm run start:dev

# 生产环境
$ npm run start:prod
```
